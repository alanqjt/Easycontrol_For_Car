## 已知问题

一.音频采集，无论是导航，还是音乐，走的都是媒体通道，会导致导航的声音走车机的媒体通道出来（媒体音量控制声音大小），

情况1：手机导航，手机音乐，车机执行播放,一切正常

情况2：手机导航，车机音乐，车机执行播放，因为混用了同一个通道，但是有两个采集端，会出现某一方的音量大小不相等情况，一方大，另一方小的情况。

情况3：车机导航，手机音乐，车机执行播放,一切正常
