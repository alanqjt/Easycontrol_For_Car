## 简介
在易控车机版的基础上

1.UI进行优化

2.修复多路投屏采集音频问题

3.优化导航语音播报卡顿吞字问题

4.优化在投屏的时候，车机进入360全景或者倒车影像的时候，自动隐藏投屏
