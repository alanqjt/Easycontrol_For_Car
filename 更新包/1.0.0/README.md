# 易控车机版 1.0.0

## 版本说明

本版本基于易控车机版 `1.6.0` 开发，主要面向车机使用场景，对车机 UI、多窗口投屏、音频分路和导航播报体验进行优化。

## 主要优化

- 优化车机端 UI，使首页、设备卡片和设置界面更适合横屏及大屏操作。
- 优化多窗口投屏能力，支持同时显示多个投屏窗口。
- 优化音频采集、传输和播放流程，为导航音频与媒体音频分路提供基础能力。
- 优化导航播报体验，减少播报延迟及开头内容被截断的问题。

## 已知问题

- 当前音频仍只有一路采集和传输。
- 手机导航声音与音乐等媒体声音会混合后统一发送到车机。
- 导航声音在车机端使用媒体通道播放，而不是独立的导航播报通道，因此可能出现媒体音量、音频焦点和播报优先级不符合预期的情况。


## 典型场景

### 手机导航 + 手机音乐

导航进入车机导航通道，音乐进入媒体通道。两路可以同时播放并由车机分别处理。

### 手机导航 + 车机音乐

手机导航使用导航通道，车机音乐继续使用媒体通道，降低两边音量互相影响的概率。

### 车机导航 + 手机音乐

车机导航保持原有导航通道，手机音乐通过媒体通道播放。

### 两个应用流转 + 一个直接投屏

导航和音乐应用流转分别拥有对应音频角色，直接投屏作为备用。关闭其中一个应用流转时，只迁移对应角色，不影响另一种声音。
